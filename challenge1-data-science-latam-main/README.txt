# Proyecto de Análisis de Ventas para la Toma de Decisiones

Este proyecto analiza datos de ventas de cuatro tiendas para ayudar al Sr. Juan a tomar decisiones informadas sobre dónde enfocar sus esfuerzos de ventas.

## Descripción del Proyecto

El objetivo principal es analizar los datos de ventas de cuatro tiendas diferentes, incluyendo información sobre facturación, categorías de productos, calificaciones de clientes, costos de envío y productos más/menos vendidos.  Se utilizarán visualizaciones y análisis estadísticos para identificar patrones y tendencias en los datos y proporcionar recomendaciones basadas en datos para el Sr. Juan.


## Metodología

1. **Recopilación de datos:** Los datos de ventas se recopilan de archivos CSV ubicados en URLs públicas de GitHub.
2. **Limpieza y preprocesamiento de datos:** Se realiza la limpieza de los datos para asegurar la calidad de la información.
3. **Análisis exploratorio de datos:** Se realiza un análisis exploratorio de los datos para entender la estructura de los datos, descubrir patrones y relaciones entre las variables. 
4. **Análisis de ventas:** Se calcula el total de ventas por tienda, la distribución de las ventas por categoría, la calificación promedio de clientes, el costo promedio de envío y se identifica los productos más y menos vendidos. Se generan gráficos para visualizar estos datos.
5. **Análisis geoespacial:** Se utiliza la información geográfica (latitud y longitud) de las ventas para crear mapas de calor y visualizar la distribución geográfica de las ventas. Además, se realiza un análisis de las ventas por regiones geográficas.
6. **Generación de informes:**  Se genera un informe con los hallazgos clave y recomendaciones para el Sr. Juan.


## Tecnologías Utilizadas

* **Python:** Lenguaje de programación principal para el análisis y visualización de datos.
* **Pandas:** Biblioteca para la manipulación y análisis de datos.
* **Matplotlib y Seaborn:** Bibliotecas para la visualización de datos.


## Instrucciones de Ejecución

1.  Clonar el repositorio.
2.  Ejecutar el notebook de Jupyter proporcionado.
3.  Los resultados del análisis se mostrarán en el notebook, incluyendo gráficos y tablas.

## Resultados Esperados

El proyecto generará un informe completo con información útil para ayudar al Sr. Juan a decidir dónde enfocar sus esfuerzos de venta. El análisis incluirá:

* **Total de ventas por tienda:** Un gráfico de barras que representa las ventas totales por tienda.
* **Ventas por categoría de producto:**  Un gráfico circular que representa la distribución porcentual de las ventas por categoría.
* **Calificación promedio por tienda:** Un gráfico de barras que representa la calificación promedio de los clientes por tienda.
* **Costo de envío promedio por tienda:** Un gráfico que representa el costo promedio de envío por tienda.
* **Productos más y menos vendidos:** Gráficos de barras para visualizar los productos más y menos vendidos por cada tienda.
* **Análisis geoespacial:** Mapas de calor y gráficos para visualizar la distribución geográfica de las ventas y el promedio de ventas por región.

El informe también proporcionará una recomendación final sobre dónde debería enfocar sus esfuerzos de venta el Sr. Juan, junto con consideraciones adicionales.
